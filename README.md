# 🍏 Applepen Retail Data Processor

Скрипт для обработки данных торговой сети **Applepen** (яблоки и карандаши).

## 📌 Что делает

Проект решает 3 задачи:

### 1. 📅 Daily inventory

Восстанавливает остатки товара **на конец каждого дня**.

Файл:

```
output/MS-{store}-daily_inventory.csv
```

Формат:

```
date,apple,pen
2006-01-01,14105,770
```

---

### 2. 🕵️ Monthly stolen

Оценивает количество **украденного товара за месяц**.

Файл:

```
output/MS-{store}-monthly_stolen.csv
```

---

### 3. 📊 Aggregation (state + year)

Агрегирует продажи и кражи по **штату и году**.

Файл:

```
output/sales_stolen_by_state_year.csv
```

Формат:

```
year,state,apple_sold,apple_stolen,pen_sold,pen_stolen
2006,MS,100,5,50,2
```

## ⚙️ Логика

* Продажи → считаются из логов (`sell`)
* Поставки → добавляются к складу (`supply`)
* Инвентарь → фиксируется раз в месяц (`inventory`)
* Кражи = расхождение между расчётным и фактическим остатком

Формула:

```
stolen = prev + supply - sales - actual_inventory
```

---

## 🧠 Особенности

* Первый месяц считается **без краж** (для восстановления начального остатка)
* Отрицательные кражи → обрезаются до 0
* Все расчёты идут **по дням внутри месяца**
